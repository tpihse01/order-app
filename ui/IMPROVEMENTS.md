# 코드 개선 사항

## 🔴 중요 (즉시 수정 권장)

### 1. 재고 관리 로직 추가
- **문제**: 주문 완료 시 재고가 차감되지 않음
- **위치**: `App.jsx`의 `addOrder` 함수
- **해결**: 주문 시 각 상품의 재고를 차감하는 로직 추가 필요

### 2. 장바구니 Key 개선
- **문제**: `OrderPage.jsx`에서 `key={index}` 사용
- **위치**: Line 259
- **해결**: 고유 ID 조합 사용 (예: `key={`${item.productId}-${item.selectedOptions.map(o => o.id).join('-')}`}`)

### 3. 재고 검증 추가
- **문제**: 장바구니 수량 증가 시 재고 확인 없음
- **위치**: `OrderPage.jsx`의 `updateCartItemQuantity` 함수
- **해결**: 수량 증가 시 재고 확인 로직 추가
